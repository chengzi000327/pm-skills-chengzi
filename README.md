# pm-skills-chengzi

个人 Claude Code PM 技能合集。

## 安装

```
/plugin install pm-skills-chengzi
```

## 包含内容

- **write-prd**：自定义 PRD 写作流程，通过链式调用多个 skill 生成决策级产品需求文档
- **prd-development**：结构化 PRD 构建 workflow，连接问题定义、用户研究、解决方案与成功标准

## 使用方式

| 命令 | 说明 |
|------|------|
| `/write-prd` | 从功能描述或问题陈述生成完整 PRD |
